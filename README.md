# agent-demo

A minimal Snake game for the browser. Clone or download this repo, run a local server, and play on localhost.

## Quick Start

From the project directory, run:

```bash
python3 -m http.server 8000
```

Then open:

- <http://localhost:8000>

## Controls

- Arrow keys: move
- Space: pause / resume
- Enter: restart after game over

## Project Structure

- `index.html`: page structure
- `style.css`: game styling
- `game.js`: Snake game logic
